## Que es Next.js

Next.js es un framework de React que simplifica el desarrollo de aplicaciones web modernas, ofreciendo renderizado del lado del servidor (SSR), generación de sitios estáticos (SSG) y un sistema de enrutamiento basado en archivos. Esta guía te ayudará a crear tu primera aplicación Next.js y a comprender sus características principales.

## Conocimientos previos necesarios

Para aprovechar al máximo Next.js, es útil tener conocimientos básicos en:

- **HTML**: Estructura de páginas web.
- **CSS**: Estilización de interfaces.
- **JavaScript**: Lógica del lado del cliente y servidor.
- **React**: Componentes, estado y propiedades.
